# HOW TO USE

### **1. Clone this repo**

### **2. Run following**

- #### cd vue-webpack-boilerplate (or change the name of the folder to what you want)

- #### npm install

- #### npm run start (for Development)

- #### npm run build (for Production)

#### Source files exist in /src, and will be complied at /dist
